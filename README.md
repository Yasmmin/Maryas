Uma aplicação web interativa onde casais podem enviar fotos e receber avaliações sinceras um do outro de forma divertida e intuitiva, com mecânica inspirada em swipe (estilo Tinder).

- 📸 Upload de fotos com preview em tempo real  
- 🔄 Avaliação por swipe (like 👍 ou dislike 👎)  
- 🧑‍🤝‍🧑 Sistema de dois usuários (casal)  
- 💬 Identificação de quem enviou cada foto  
- 🖼️ Galeria com fotos enviadas  
- ⚡ Interface moderna com animações suaves (Framer Motion)
